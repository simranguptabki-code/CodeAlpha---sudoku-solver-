
# Sudoku Solver (C++)

## Overview
This project is a Sudoku Solver developed in C++. It uses the Backtracking Algorithm to solve a 9×9 Sudoku puzzle.

## Features
- Solves any valid Sudoku puzzle
- Uses Backtracking Algorithm
- Displays solved Sudoku
- Checks row, column and 3×3 box validity

## Technologies Used
- C++
- Arrays
- Recursion
- Backtracking

## How to Run
1. Compile the program using any C++ compiler.
2. Enter the Sudoku puzzle.
3. The program will display the solved Sudoku.
